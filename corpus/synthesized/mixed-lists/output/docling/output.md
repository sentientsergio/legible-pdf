## 4 Programming Language Terms

Compiler theory distinguishes several program transformation roles.

Compiler A program that translates source code into executable code.

Interpreter A program that executes source code line by line.

Linker A program that combines compiled object files into an executable.

These roles often overlap in modern toolchains but remain conceptually distinct.
